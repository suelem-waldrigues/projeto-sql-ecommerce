# 🗄️ Projeto SQL - Sistema de Banco de Dados

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar SQL e banco de dados relacional.

O sistema simula um cenário de e-commerce, contendo tabelas como clientes, produtos, categorias, fornecedores, pedidos e funcionários.

---

## 🧠 Conceitos Utilizados

* CREATE TABLE
* INSERT INTO
* SELECT
* UPDATE
* DELETE
* WHERE
* BETWEEN
* ORDER BY
* FOREIGN KEY

---

## 🛠️ Tecnologias

* SQL
* SQLite

---

## 📂 Estrutura

* database.sql → contém todo o banco de dados

---

## 🚀 Como executar

1. Abra o SQLiteOnline ou outro banco
2. Copie o conteúdo do arquivo `database.sql`
3. Exe
